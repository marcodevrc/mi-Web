Web 2.0

La Web Interactiva

La Web 2.0 representó una evolución significativa respecto a la Web 1.0 al introducir la posibilidad de interacción y participación activa de los usuarios. Surgió a mediados de los años 2000 y permitió el desarrollo de aplicaciones web dinámicas. Durante esta etapa se popularizaron las redes sociales, los blogs y plataformas colaborativas como Wikipedia. Se caracteriza por facilitar la creación de contenido generado por los usuarios, marcando el inicio de una web más social y accesible.
